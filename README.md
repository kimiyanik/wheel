# wheel

A GDPR-friendly Global AI Community wheel-of-names experience with mobile QR join. Scan the QR code to open the join page, add your name, and let the host spin the wheel to pick winners.

## Run locally

1. `npm install`
2. `npm run dev`
3. Open `http://localhost:4173`

## Firebase setup

1. Copy `firebase-config.example.js` to `firebase-config.js`.
2. Fill in your Firebase Realtime Database config.
3. Ensure your database rules allow writes for this demo or authenticate accordingly.

## Pages URL

- Presenter host page: `https://kimiyanik.github.io/wheel/`
- Mobile join page: `https://kimiyanik.github.io/wheel/#join`
